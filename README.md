#  Virtual Oscilloscope

A fully software-based virtual oscilloscope that uses your laptop's
microphone to capture and display audio signals in real-time.

##  Features

### Signal Display
- Real-time waveform display (time domain)
- FFT spectrum analyzer (frequency domain)
- Combined view (both simultaneously)
- Phosphor persistence (CRT afterglow effect)
- CRT-style glow rendering

### Timebase Controls
- Adjustable Time/Division
- Horizontal position control

###  Channel Controls
- Adjustable Volts/Division (vertical zoom)
- Vertical offset
- AC/DC coupling

###  Trigger System
- Auto, Normal, and Single trigger modes
- Rising and Falling edge detection
- Adjustable trigger threshold
- Visual trigger level indicator

###  Measurements
- Frequency (Hz)
- Period (ms)
- Peak-to-peak voltage (Vpp)
- RMS voltage (Vrms)
- Maximum voltage (Vmax)
- Minimum voltage (Vmin)
- DC average

###  Display Customization
- 6 trace colors (Green, Cyan, Yellow, Orange, Magenta, White)
- Adjustable line width
- Adjustable grid brightness
- Phosphor persistence control

###  Tools
- Freeze/Unfreeze waveform
- Screenshot export (PNG)
- Waveform data export (CSV)
- Audio recording export (WAV)
- Hover tooltip (time and amplitude)
